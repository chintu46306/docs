---
title: 'Get All Notifications'
api: 'GET /api/admin/notifications'
description: 'Retrieve all notifications (admin only)'
authentication: 'Bearer Token'
---

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Response

<ResponseField name="notifications" type="array">
  Array of notification objects containing:
  - `sender`: User ID of the sender
  - `recipients`: Array of recipient User IDs
  - `message`: Content of the notification
  - `isCritical`: Boolean indicating if notification is critical
  - `timestamp`: Date when notification was created
  - `status`: One of `"pending"`, `"delivered"`, or `"queued"`
  - `deliveredAt`: Date when notification was delivered (if applicable)
  - `scheduledFor`: Date when notification is scheduled for (if applicable)
</ResponseField>

<ResponseExample>
```json
[
  {
    "sender": "507f1f77bcf86cd799439011",
    "recipients": ["507f1f77bcf86cd799439012", "507f1f77bcf86cd799439013"],
    "message": "Important update",
    "isCritical": false,
    "timestamp": "2024-03-21T10:00:00Z",
    "status": "delivered",
    "deliveredAt": "2024-03-21T10:00:01Z"
  }
]
```
</ResponseExample>
